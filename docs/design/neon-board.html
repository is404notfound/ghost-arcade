Ghost Dash — 네온 세기말 시안 3종

1040 × 480 (19.5:9) · 좌측 고정 플레이어 · 도형 기반 · 60fps WebView · 메테오/배경은 순수 장식(결정론 보존)

게임플레이 색 의미(3종 공통): 플레이어 고스트(라이벌 리플레이) 연료통(회복·보상) 건물=장애물(위험)
핵심 제약 — 떨어지는 메테오·패럴랙스 스카이라인·창문 깜빡임은 전부 렌더 전용 장식입니다. 충돌/게임영향이 있으면 sim에 들어가야 하고 → 결정론이 깨져 → 고스트 리플레이·서버검증·토큰보상 thesis가 전부 무너집니다. 네온 세기말의 "공기"는 마음껏, 단 게임에 닿으면 안 됩니다.
A

세기말 노을 · Synthwave

후드 라이더가 네온 오토바이로 질주. 발로 뛰는 헤일로 고스트(죽은 라이벌)를 추월. 배경 속도선이 실제로 흐르고(움직이는 배경) 메테오가 낙하 — 속도감 중심. ※ 브라우저에서 애니메이션 재생됨.

팔레트

#170a2e sky-top #6b1248 sky-low #ff5fa2 sun #36f9f6 grid/edge

건물 처리

  • 다크필(#0d0618) + 시안 외곽선 + 창문 점 2열
  • 안테나로 키 작은 32px도 "타워"로 인식
  • 창문색 마젠타/시안 교차 → 위험감

비용 / 리스크

글로우 예산: 높음. 바닥 그리드 + 노을 + 메테오 글로우가 겹침. 저사양 폰에선 그리드 라인 수·블룸 반경을 줄여야 60fps.

B

방사능 폐허 · Toxic Acid

적대적 종말. 독성 그린 하늘 + 부서진 건물. 플레이어(쿨 시안)와 세계(독 그린)의 대비가 최대 → 가독성 최상.

팔레트

#03100b sky-top #0c3a22 sky-low #9dff5a acid #ffb14d hazard

건물 처리

  • 창문 일부 소실 → "폐허" 인상
  • 그린 창 + 드물게 앰버(#ffb14d) 위험 점멸
  • 바닥에 방사능 웅덩이 글로우

비용 / 리스크

글로우 예산: 중. 단색(그린) 위주라 A보다 가벼움. 단, 그린 일색이라 연료통(블루)·플레이어(시안) 대비를 의도적으로 살려야 함.

C

정전 미니멀 · Blackout Mono-neon

가장 절제된 종말. 순흑 + 단일 시안 외곽선. DESIGN.md "미니멀" 원칙에 가장 충실하고 60fps 예산 최저.

팔레트

#050507 void #19e0ff neon #ff3b3b hazard #d6fbff meteor

건물 처리

  • 순흑 실루엣 + 단일 시안 외곽선만
  • 위험 신호 = 상단 4px 레드 스트라이프
  • 창문 1~2개만 점등(노이즈 최소)

비용 / 리스크

글로우 예산: 낮음(안전). 메테오 2개·단색·그리드 없음 → 저사양 60fps 여유. 리스크: 너무 비어 보일 수 있음(허전함 vs 절제).

이 시안에서 봐야 할 것 — ① 종말의 "공기"가 게임 가독성을 해치지 않는지, ② 32px 건물이 "건물"로 읽히는지, ③ 플레이어/고스트/연료통/위험 4색이 한눈에 구분되는지, ④ 글로우 비용이 폰에서 감당되는지.

피드백 주실 것 — 선호 방향(A/B/C 또는 조합, 예: "C 배경 + A 건물"), 색 조정, 메테오 밀도(많이/적게), 건물을 더 높고 가늘게 보이게 할지(배경 데코 타워 강화) 등.
