- 고정 타임스텝 어큐뮬레이터 (렌더 fps와 무관하게 시뮬 스텝 고정)

## Failure modes

| 코드패스 | 프로덕션 실패 시나리오 | 테스트 | 에러 핸들링 | 유저 가시성 |
|---|---|---|---|---|
| 시뮬 step | 부동소수점/엔진 차이로 재생 불일치 | 골든 리플레이 (T4) | 버전 태그 제외 처리 | 고스트 미표시 (명시적) |
| 입력 기록 | 백그라운드 전환 중 로그 손상 | T7 | 일시정지 시 로그 플러시 | 재개 화면 |
| PostHog | 차단/타임아웃 | 격리 테스트 (T6) | try/catch 격리 | 영향 없음 (silent — 의도됨) |
| 렌더 루프 | 저사양 GC 멈칫 | 수동 (저사양 기기) | 풀링/제로 할당 (예방) | 프레임 드랍 |
| localStorage | 용량 초과/비활성 | — | try/catch + 메모리 폴백 필요 | 점수 미저장 |

**Critical gap: 0건** — localStorage 실패는 T6 구현 시 try/catch + 메모리 폴백 포함할 것 (위 표에 명시).
