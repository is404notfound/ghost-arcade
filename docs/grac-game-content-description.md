# 게임물내용설명서 — Ghost Arcade

작성일: 2026-07-13  
용도: 게임물관리위원회(GRAC) 등급분류 신청용  
앱 식별: `ghost-runner` (앱인토스 appName)  
표시명: Ghost Arcade (콘솔 표시명과 다르면 콘솔 값으로 통일)

---

## 1. 줄거리(시나리오) 요약

어느 날 지구에 운석이 떨어지기 시작하며 인류는 멸종 위기에 처한다. 재앙의 비밀을 밝히려던 수많은 영웅들은 도로 위에서 쓰러졌고, 주인공만 남았다. 주인공은 앞선 영웅들이 달렸던 길을 따라, 그들의 흔적(고스트·부서진 바이크 잔해)을 밟으며 끝없는 도로를 홀로 질주한다.

게임은 이야기 분기·선택지·엔딩 분기가 없는 **엔들리스 러너**이며, 인트로에서 위 세계관을 짧게 제시한 뒤 즉시 플레이로 들어간다.

**인트로 고정 카피**

> 종말이 다가오자, 수많은 이들이 그 비밀을 쫓다 쓰러졌다.  
> 마지막 등불, 그 흔적을 밟으며 다시 달린다.

---

## 2. 주요 캐릭터

| 구분 | 설명 |
|------|------|
| **주인공(플레이어)** | 시안 네온 후드와 바이크를 탄 라이더. 플레이어가 조작하며, 장애물을 뛰어넘고 연료통을 모아 최대한 멀리 달린다. |
| **고스트** | 다른 이용자의 **과거 주행 기록**이 반투명 유령으로 재생되는 존재. 실시간으로 상대를 공격하거나 피격하지 않으며, 같은 도로를 비동기로 달리는 경쟁 연출용이다. 골드 헤일로는 쓰러진 영웅의 표식이다. |
| **장애물** | 도로상의 건물·잔해 등. 부딪히면 체력(HP)이 감소한다. 지능형 NPC·대화 상대가 아니다. |

이용자 간 실시간 대결(PvP/PK/RvR), 강제 전투, 아이템 약탈은 없다.

---

## 3. 주요 아이템 (리스트·조합 시스템)

### 3-1. 아이템 리스트

| 아이템명 | 효과 | 획득 방식 |
|----------|------|-----------|
| **연료통(포션)** | 체력(HP) 회복 | 주행 중 도로 위·공중에 출현. 접촉 시 획득 |
| **피버(버프)** | 일정 시간 스크롤 가속, 피버 중 탭 시 소량 HP 회복, 점프 제약 완화 | 장애물 연속 통과(콤보)로 발동. 별도 소지 아이템이 아님 |

그 외 무기·방어구·소모품·유료 아이템·가챠 상자는 **없다**.

### 3-2. 아이템 조합·제작·강화 시스템

**없음.**

아이템을 합치거나, 재료로 제작·강화·합성하는 시스템은 구현되어 있지 않다. 연료통은 획득 즉시 효과가 적용되며 인벤토리·장비란이 없다.

---

## 4. 전투장면에 대한 설명

본 게임에는 **이용자 대 이용자(또는 NPC)와의 전투 장면이 없다.**  
장르는 횡스크롤 **장애물 회피형 러너**이며, 공격·스킬·무기 전투 UI가 없다.

충돌·피격에 해당하는 진행은 다음과 같다.

1. 플레이어가 도로를 자동 전진한다.
2. 전방에 장애물이 나타나면 **점프(최대 2단)** 로 회피한다.
3. 장애물과 충돌하면 HP가 감소한다.
4. 평시 HP는 시간에 따라 서서히 감소하며, 연료통으로 회복할 수 있다.
5. **피버** 중에는 가속·회복 보조가 있고, 충돌 판정이 완화되는 구간이 있다.
6. HP가 0이 되면 해당 판이 **게임오버**되며, 최종 거리(m)가 점수·랭킹에 반영된다.

폭력 연출(피·고어·살해 연출) 없는 아케이드식 충돌·회피이며, 전투 승리/패배에 따른 아이템 강탈·경험치 약탈은 없다.

---

## 5. 게임 설치 / 조작방법

### 5-1. 설치(이용) 방법

- **플랫폼:** 토스 앱 내 **앱인토스(미니앱)** 로 제공 (WebView 기반).
- **앱 식별:** `ghost-runner` (콘솔 appName 기준).
- **설치:** 별도 APK/IPA 설치가 아니라, 토스 앱에서 해당 미니앱을 실행하면 곧바로 플레이 가능하다.
- **권한:** 게임 진행에 특수 단말 권한(카메라·마이크·위치 등)을 요구하지 않는다.
- **과금/경품:** 1차 제공 기준 무료 플레이. 경품 배출·현금 환전 없음.

### 5-2. 조작방법

| 입력 | 동작 |
|------|------|
| 화면 탭 / 마우스 클릭 | 점프 (지상 1단 + 공중 1단, 최대 2단) |
| `Space` / `↑` | 점프 (동일) |
| 일시정지 버튼 | 일시정지 / 재개 |
| 다시하기 버튼 | 일시정지 중 현재 판 재시작 |
| 음소거 버튼 | BGM·효과음 on/off |
| 전체화면 버튼 | 전체화면 토글 (지원 환경) |

조작 학습은 인트로 이후 **howto 튜토리얼**(탭 점프 → 1단/2단 점프 → 장애물 회피·연료통)으로 안내한다.  
조준·스킬 단축키·채팅 단축키 등 추가 Hotkey는 없다.

### 5-3. 점수·종료

- **점수:** 달린 거리(m). 일일·주간 랭킹에 기록될 수 있다.
- **종료 조건:** HP 0(게임오버). 이용자가 일시정지 후 다시하기로 판을 끝낼 수도 있다.
- **이용자간 대결:** 실시간 PvP 없음. 고스트·랭킹은 과거 기록의 비동기 비교만 해당.

---

## 6. 이용자간 대결 (참고 — 신청서 사례란)

**해당 없음.**

본 게임은 이용자 간 실시간 대결(PvP, PK, RvR 등)이 없다.  
다른 이용자의 과거 주행 기록이 ‘고스트’로 화면에 재생되고, 일일·주간 거리 랭킹이 표시되는 비동기 기록 비교만 있으며, 대결 승패에 따른 보상·손실(아이템·재화 획득·몰수 등)은 없다.

---

## 7. 서류용 한 줄 요약

> 운석 재앙 이후 홀로 도로를 질주하는 후드 라이더의 엔들리스 러너. 탭/점프만으로 장애물을 피하고 연료통으로 HP를 유지하며, 다른 이용자의 과거 기록이 고스트로 재생된다. 전투·아이템 조합·경품·실시간 PvP 없음.

---

## 8. 첨부 스크린샷 권장 목록

신청서에 장면을 붙일 때 아래 구성을 권장한다.

1. 인트로(스토리 슬라이드)
2. 본편 런(장애물·고스트·HP/거리 HUD)
3. 피버 발동 연출
4. 게임오버·일일/주간 랭킹
