# Phase 0 관찰 노트

목적: "이 컨셉이 재미있나(자발적 재시도)"를 싸게 검증. 코드는 폐기 전제 —
여기 적힌 *학습*만 Phase 1(src/sim/) 재작성으로 가져간다.

기록 항목: ① 자발적 재시도 여부 ② 첫 사망까지 거리/시간 ③ 재시도 직전 발언
(숙달 욕구 "거의 됐는데" vs 새것 호기심 "신기하네" 구분).

## 프로토 #1 — canvas-tap

탭 점프 러너 + 에스컬레이션 + 니어미스 + 체력/포션 + 거리(M) 점수 + 2단 점프.

### 테스터 #1 (외부, 맥락 없이 건넴)
- ① 자발적 재시도: **O** — "한번만 더 해볼게요" 후 여러 판
- ② 첫 사망: 400~500M (적당히 늦음 = 2초컷 아님, 학습 곡선 OK)
- ③ 재시도 이유: **숙달 욕구** (여러 판 반복 = 새것 호기심 아님). 강한 신호.
- 기타 발언:
  - **"막대(장애물)는 랜덤이냐?"** → 핵심 확증. 테스터가 본능적으로 패턴을
    외우려 함. 결정론/시드 베팅(D10, 시드 공유 TODO)이 옳다는 살아있는 증거.
    순수 랜덤이면 "불공평"으로 이탈, 시드 고정이면 숙달 가능 + 고스트 공정성 확보.
  - "장애물 피하기 힘들다" → 난이도 튜닝 (단, 3단 점프 버그 있던 상태의 평가).
  - "점프가 3단까지 된다" → 버그였음. 2단 의도. 수정 완료
    (velocity.y >= 0 착지 조건 추가). 수정으로 난이도 상승 — 추가 상향 금지.

### 미해결 / Phase 1로 가져갈 설계 신호
- 포션 위치가 애매함 — 튜닝이 아니라 위험/보상 결정이 실제로 작동하는지 설계 신호.
- 장애물 시드/결정론: 테스터가 패턴 학습을 시도 → 시드 고정 방향에 무게.
  (TODOS.md 고스트 시드 공유 결정과 직결)

### 테스터 #2, #3 (외부)
- 3명 전원 자발적으로 최소 2판씩 플레이. (n=3, 자발적 재시도 100%)
- Phase 0 게이트: 외부 다수가 시키지 않고 재시도 → **통과**.

### 현황
- **Phase 0 결론: canvas-tap = 승자 컨셉.** 외부 n=3 자발적 재시도로 검증.
- 코드는 폐기 전제. 아래 학습만 Phase 1(src/sim/)로 가져간다:
  1. 장애물 시드 고정 = 패턴 학습 가능 (테스터 전원이 외우려 함). 결정론 코어의 근거.
  2. 에스컬레이션 곡선에 상한 필요 (속도+간격 이중 가속 → 벽). 튜닝 대상.
  3. 포션 위치/위험보상 재설계 여지.
  4. 피격 후 무적 시간 확대 = 죽음의 소용돌이 완화 (속도 감속 안 함 — 인센티브 보존).
